# Example Beatmap: Simple 4/4 Beat

This is a minimal example beatmap for testing and learning.

## How to Use

1. Place your audio file (`simple_beat.mp3`) in the same directory as this file
2. Load this beatmap in BeatSight
3. Play!

## Beatmap Details

- **Title**: Simple Practice Beat
- **BPM**: 120
- **Difficulty**: 2.0 (Easy)
- **Duration**: ~10 seconds
- **Drum Parts**: Kick, Snare, Hi-hat

## Pattern

```
Beat:    1   e   &   a   2   e   &   a   3   e   &   a   4   e   &   a
Kick:    X               X               X               X
Snare:                                  X                               X
Hi-hat:  X       X       X       X       X       X       X       X
```
